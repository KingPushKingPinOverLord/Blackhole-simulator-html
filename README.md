# Simple Black Hole Visualizer

A clean, lightweight visual effect of a black hole built entirely with the **HTML5 Canvas API** and vanilla **JavaScript**. There are no complex physics equations or heavy configuration panels here—just a cool-looking space visual running smoothly in your browser.

---

## 🎨 What It Is

* **Pure Eye Candy:** A simple, hypnotic particle animation looping around a dark central void.
* **Lightweight & Fast:** Built using native web technologies so it loads instantly.
* **Zero Setup:** No libraries, frameworks, or installations required. 

---

## 🚀 How to Run

1. Download or clone this project folder.
2. Open `index.html` directly in any modern web browser (Chrome, Safari, Firefox, Edge).
3. Sit back and watch the visual.
